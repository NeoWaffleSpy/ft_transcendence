IF YOU RUN DOCKERLESS:
	1. make BDD in Containers/ folder to start postgresql database

	2. Backend dependencies: (you have to be in the Backend folder):
		npm install @nestjs/cli@latest && \
		npm install class-transformer@latest --save && \
		npm install --save dotenv@latest && \
		npm install --save @nestjs/typeorm@latest pg && \
		npm install --save @nestjs/passport@latest passport@latest && \
		npm install --save-dev @types/passport-local@latest && \
		npm install --save @nestjs/jwt@latest passport-jwt@latest && \
		npm install --save @nestjs/websockets@latest @nestjs/platform-socket.io@latest && \
		npm install --save @nestjs/axios@latest
	Then run with npm run start:dev

	3. Frontend dependencies: (you have to be in the Frontend folder):
		npm install @angular/cli && \
		npm install axios && \
		npm install jquery && \
	Then run with npm run start
